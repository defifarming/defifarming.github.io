Frequently asked questions

Please find the answers to the most frequently asked questions below.

What is Scrum Network blockchain?

Scrum Network Chain is to bring programmability and interoperability to EVM compatible Chain. In order to embrace the existing Tools and Technology Scrum Network Chain offer several features that existing Chain don’t offer, Scrum Network Chain will bring huge benefits by staying compatible with all the existing smart contracts on Ethereum, Binance smartchain and several EVM chain, Developer can use all Ethereum tooling on Scrum Network chain.

Scrum Network Consensus algorithm??

Scrum Network uses The Proof-of-Staked-Authority (PoSA) consensus algorithm it is a hybrid between the Proof-of-Staking and Proof-of-Authority consensus mechanism. The PoSA consensus model supports a shorter block time and Zero transaction Fees than, for example, the PoW consensus model that Ethereum (ETH) uses. On Scrum Network chain validators are participants who stake a certain number of SN. When they propose a valid block, they receive $Scrum as a reward for the transactions contained in that block. Validator are determined by ranking according to the Amount of (SN) self-stake and delegated by delegator. This is determined once a day and the set of all validators is stored separately.

Purpose of SN?

The SN Coin has a number of use cases within the Scrum Network chain. It plays an essential role for a Securing Network.

What is staking reward?

The staking rewards for SN are variable minimum 14% APY. The rewards on Scrum Network are dynamic and decrease as the staking participation increases.

What is validator?

Validators are the backbone of the Scrum Network. They play critical role on Scrum Network By processing transactions and participating in consensus.

Who can be validator?

Anyone can be validator as long as they have minimum self-stake SN.

What is Delegator?

Scrum Network uses Proof of Stake Authority Consensus algorithm so there is validator and Delegator to secure the Scrum Network. Delegator gets incentivize by staking their SN and vote to potential validator.
